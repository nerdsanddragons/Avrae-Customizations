These three aliases allow you to roll on the Random Treasure, Hoard, and Magic Item tables in the DMG on [Page 136](https://www.dndbeyond.com/sources/dmg/treasure#RandomTreasure).

`!treasure [cr]`
`!hoard [cr]`
`!mitable [A-I] [#|dice]`
